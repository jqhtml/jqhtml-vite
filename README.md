# @jqhtml/vite-plugin

Vite plugin for compiling `.jqhtml` templates during build.

This is a helper module for [jqhtml Laravel integration](https://github.com/jqhtml/jqhtml-laravel). See that repository for installation and usage instructions.

For full jqhtml documentation, visit https://jqhtml.org/
